# Part 1 - Data Audit, EDA & Business Understanding

## How to run

```bash
python -m pip install -r requirements.txt
jupyter notebook eda_audit.ipynb
```

The generated reports and charts are already included in this repository. The notebook reads from `data/`.

## Outputs

- `eda_audit.ipynb`
- `data_quality_report.md`
- `business_memo.md`
- `charts/` with eight dataset-backed charts
